Software requirement

  1. NVM ( node version manager )
    Install NVM from here https://github.com/nvm-sh/nvm
    Once you have installed , use it to install node version 18

    nvm install v18

  2. Install Truffle
    https://archive.trufflesuite.com/docs/truffle/how-to/install/

    npm install -g truffle

  3. Install Metamask
    https://metamask.io/download/

  4. Download Ganache
    https://archive.trufflesuite.com/ganache/

Once all 4 are install , deploy the first contract using following commands
( Make sure that Ganache is running )

git clone https://github.com/nautical/lpu_307
cd Supplement/truffle_sample_project
npm i
truffle migrate
